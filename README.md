# Evergruv

A color theme based on [@sainnhe](https://github.com/sainnhe)'s excellent [Everforest Light](https://github.com/sainnhe/everforest) for the light variant, and [Gruvbox Material Dark](https://github.com/sainnhe/gruvbox-material) for the dark variant. The color palette and its application has been altered slightly. If you encounter issues with this theme, please [report them in the GitHub repository](https://github.com/georglauterbach/evergruv/issues).

## Theme Applications

1. Wallpapers: Published to [`georglauterbach/desktop`](https://github.com/georglauterbach/desktop) in [`data/.config/sway/theme/`](https://github.com/georglauterbach/desktop/tree/main/data/.config/sway/theme)
2. VS Code Theme: Published to the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=georglauterbach.evergruv) and the [Open VSX Registry](https://open-vsx.org/extension/georglauterbach/evergruv)
3. `btop` Theme: Published to [`georglauterbach/hermes`](https://github.com/georglauterbach/hermes) in [`data/config/btop/themes/`](https://github.com/georglauterbach/hermes/tree/main/data/config/btop/themes)
